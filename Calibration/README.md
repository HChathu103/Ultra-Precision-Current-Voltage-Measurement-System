# Calibration System

## Description
Python scripts used to calibrate voltage and current measurements.

## Files
- voltage_calibration.py
- current_calibration.py

## Purpose
- Reduce measurement error
- Improve system accuracy

## Method
- Compare measured vs actual values
- Apply correction formula

## Requirements
- Python 3.x
- numpy
- matplotlib

## How to Run
```bash
python voltage_calibration.py
